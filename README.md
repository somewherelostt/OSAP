# OSAP

Personal Agent OS — A mobile-first AI agent platform with persistent memory.

## Features

- **Persistent Memory** — Your agent remembers everything
- **Cross-App Execution** — Execute tasks across applications
- **Developer-First** — Built for developers who want programmable agents

## Tech Stack

- Next.js 16 (App Router)
- TailwindCSS + shadcn/ui
- Supabase
- Bun runtime
- Zustand

## Getting Started

```bash
bun install
bun run dev
```

## Routes

- `/` — Landing page
- `/home` — Dashboard
- `/tasks` — Task management
- `/memory` — Memory timeline
- `/dev` — Developer tools
- `/profile` — User settings
